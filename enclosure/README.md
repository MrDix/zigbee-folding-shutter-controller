# Enclosure

The 3D-printable enclosure for the two-board stack will be published here as **STEP** and
**STL** files once the design has been verified on real hardware.

Design goals:

- Fits a deep flush-mount wall box (Ø 60 mm class, ~50 mm clear depth) with enough space
  left behind the device for field-wiring slack
- Mounting tabs on the standard 60 mm screw spacing of the box
- Screw terminals accessible from the front for wiring before the unit is pushed in
- Openings for the sensor connectors, buttons, status LED and external antenna
